# GitToolBuzy

## Overview
GitToolBuzy is a frictionless, desktop-integrated utility designed to bridge the gap between local Git version control and remote Issue Trackers (like Jira or self-hosted alternatives). 

Traditional project management tools often force developers to break their focus, requiring context switches to a web browser to update tickets, write descriptions, or link commits. GitToolBuzy solves this by bringing issue tracking and commit management directly to your operating system's interface via a hidden, hover-triggered top bar.

## Motivation
This project was born out of a simple premise: developers do not mind writing code or making Git commits, but the administrative overhead of managing tickets can disrupt the workflow and lead to team disorganization. GitToolBuzy eliminates the friction of issue tracking by keeping the necessary tools always at hand, reducing context switching, and automating tedious documentation tasks.

## Core Features

* **Zero-Friction Interface:** Operates as a lightweight, hidden top-bar application that reveals itself smoothly upon mouse hover, keeping your workspace clean until you need it.
* **Unified Workflow:** Connects seamlessly with your Issue Tracker. View tasks, create branches directly from tickets, and automatically link Git commits to their respective issues in one fluid motion.
* **Smart Conflict Resolution:** If a `git pull` or `git push` results in a merge conflict, the tool automatically intercepts the console output and launches VS Code (or your default editor) to resolve the conflicts visually. Includes quick-action buttons to retry or abort the merge.
* **AI-Assisted Drafting (BYOK):** Features a "Bring Your Own Key" AI integration. Users can input completely unstructured text or rough ideas, and the AI will instantly format it into a clean, structured commit message or issue description, ready for quick human review and submission.
* **Cross-Platform Ready:** Designed to work efficiently on Windows (for general team use) while being adaptable to Linux environments.

## How It Works
1. **Create & Manage:** Hover over the top screen edge to reveal the GitToolBuzy menu.
2. **Select Task:** Pick an active issue from the tracker. The tool can automatically create a local Git branch linked to this task.
3. **Commit & Push:** Once your code is ready, trigger a commit. The tool prompts for the files, the message (which can be generated by AI), and whether you want to push immediately.
4. **Automated Tracking:** The remote issue tracker is automatically updated with the new commit data, keeping the whole team synchronized without anyone having to open a web browser.
